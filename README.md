# 🏭 Desafio Iniflex
 
## 📁 Estrutura
 
```
src/main/java/com/industria/
├── Pessoa.java        → classe base com nome e data de nascimento
├── Funcionario.java   → estende Pessoa, adiciona salário e função
└── Principal.java     → executa todas as operações solicitadas
```
 
## ✅ O que o projeto faz
 
- 👥 Cadastra os funcionários conforme tabela fornecida
- ❌ Remove o funcionário João da lista
- 🖨️ Imprime todos os funcionários (data em dd/MM/yyyy, salário com R$ e separador de milhar)
- 💰 Aplica aumento de 10% nos salários
- 🗂️ Agrupa funcionários por função usando Map
- 🎂 Filtra aniversariantes dos meses 10 e 12
- 👴 Exibe o funcionário mais velho com nome e idade
- 🔤 Lista funcionários em ordem alfabética
- 🧾 Calcula o total da folha salarial
- 📊 Calcula quantos salários mínimos (R$ 1.212,00) cada funcionário recebe
 
## 🚀 Como rodar
 
### 📋 Pré-requisitos
- ☕ JDK 11 ou superior → [Download](https://adoptium.net)
- 🌑 Eclipse IDE → [Download](https://www.eclipse.org/downloads/) *(recomendado)*
- 🐦 ou NetBeans → [Download](https://netbeans.apache.org/front/main/download/)
 
### 🌑 Passos no Eclipse
1. Abra o Eclipse
2. `File` → `Import` → `Existing Projects into Workspace`
3. Selecione a pasta do projeto e clique em `Finish`
4. Abra `Principal.java`, clique com o botão direito → `Run As` → `Java Application`
 
### 🐦 Passos no NetBeans
1. Abra o NetBeans
2. `File` → `Open Project`
3. Selecione a pasta do projeto
4. Abra `Principal.java` e clique em ▶️ **Run**
